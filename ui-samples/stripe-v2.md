## Design Philosophy

- **Light theme** ΓÇö light/white backgrounds with dark text
- **Single typeface** (sohne-var) ΓÇö unified, cohesive feel
- **Pill-shaped elements** ΓÇö fully rounded buttons/badges
- **Glassmorphism** ΓÇö backdrop blur effects for depth
- **Generous whitespace** ΓÇö large section gaps for breathing room
- **Accent color** ΓÇö `#533afd` used for interactive elements
